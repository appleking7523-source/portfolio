Overview

일정을 쉽고 빠르게 관리하는 모바일 중심 서비스

Planly는 사용자가 수업, 과제, 모임, 개인 일정 등을
한곳에서 확인하고 관리할 수 있도록 설계한 일정 관리 서비스입니다.
모바일 사용 환경을 중심으로 빠른 일정 확인과
간편한 입력 흐름을 목표로 제작을 해봤습니다.

Problem

기록은 많아지는데, 정작 오늘 무엇을 해야 할지는
더 불투명해집니다.

기존 서비스들은 관리해야 할 정보는 많아졌지만 일정은
여러 곳에 흩어져 있어 중요한 일을 빠르게 파악하기
어려웠습니다.

특히 오늘 해야 할 일과 완료하지 않은 일정이 명확히
구분되지 않아 우선순위를 정하는 과정에서
불편함이 느껴졌습니다.

분리된 정보

일정과 할 일이 여러 곳에 흩어져
전체 흐름 파악이 어려움

가독성 저하

완료 항목과 남은 항목의 구분이
모호해 오늘 상황이 한눈에 안 보임

등록의 번거로움

복잡한 입력 과정 때문에 즉각적인
기록을 자꾸 미루게 됨

경험의 단절

모바일(이동 중 확인)과 PC(상세 작업)
간의 최적화된 레이아웃 부재

2026 / Portfolio

Hypothesis

분리된 일정과 할 일을 통합하여
직관적인 시간 관리 환경 제공

  1. 홈 화면에서 일정 및 할 일 등을 빠르게 확인이
    가능할 수 있도록 설계 했습니다.
  2. 완료/미완료 항목을 구분해 명확한 대비를
    주었습니다.
  3. 입력 단계를 최소화 해 불필요한 시간이 들지
    않게 했습니다.
  4. 요즘 트렌드에 맞는 모바일 사용 환경을 중심
    으로 기기별 최적화된 유연한 레이아웃으로
    구성하였습니다.
